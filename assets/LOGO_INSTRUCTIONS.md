## Image Specifications
- **Filename**: `foodventure-logo.png`
- **Format**: PNG or JPG

- **File Size**: < 500KB (optimize for mobile)
- **Content**: Food collage with various dishes (noodles, spring rolls, meat, rice, etc.)

## Steps to Add Logo
1. Save Image 2 from the conversation
2. Rename it to `foodventure-logo.png`
3. Place it in the `assets/` directory
4. Delete this instruction file
5. Restart the Metro bundler: `npm start -- --reset-cache`
